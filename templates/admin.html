薪資計算規則
自動請假扣款
開啟：系統自動依請假類型扣款（無薪假全扣、半薪假扣一半），適合大多數公司。
關閉：請假扣款完全由「薪資項目」公式控制，避免與自訂公式重複扣款。
自動缺勤扣款
開啟：有排班但未打卡也無請假的日期，系統自動扣日薪。
關閉：不自動扣缺勤，由主管手動調整薪資明細處理。
自動計算扣繳稅款
開啟：月薪資超過 NT$88,501 時自動扣 5%（薪資所得扣繳，年度憑單使用）。
關閉：不自動計算，稅款由管理員手動在薪資明細中加入扣除項目。

公式說明：可用變數： base_salary（底薪） insured_salary（投保薪資） service_years（年資） actual_days（出勤天數） personal_days（事假天數） sick_days（病假天數） daily_wage（日薪）
支援條件式：值 if 條件 else 值 範例：午餐費 actual_days*70 全勤扣款 3000*(0.5 if personal_days==1 else (1 if personal_days>=2 else 0))
自訂代號：每個項目可設定代號（如 position_pay、perf_pay），代號顯示在卡片右側供計算時參照。
計算時薪範例：（本薪 + 職務 + 績效）÷ 30 ÷ 8，對照各項目的代號自行加總後計算。

關於勞基法監控
• 系統每 7 天自動從全國法規資料庫抓取勞動基準法（法規代號 N0030001）修正紀錄
• 發現新修正時，自動在「公告管理」新增高優先度公告提醒人資人員
• 可點擊「立即檢查更新」手動觸發一次即時抓取
• 公告後請至全國法規資料庫確認完整條文，本系統僅提示修正日期，不取代法律諮詢
